# [jq-ratio-buckets](https://github.com/joelpurra/jq-ratio-buckets)

Counter buckets for the [0,1] range. Useful for having the normalized distribution of ratios 0.0-1.0 (percentages 0-100%) created from other sources.



> ## ⚠️ This project has been archived
>
> No future updates are planned. Feel free to continue using it, but expect no support.



This is a package for the command-line JSON processor [`jq`](https://stedolan.github.io/jq/). Install the package in your jq project/package directory with [`jqnpm`](https://github.com/joelpurra/jqnpm):

```bash
jqnpm install joelpurra/jq-ratio-buckets
```
